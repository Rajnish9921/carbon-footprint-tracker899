# Carbon Footprint Awareness Platform

## Overview

The Carbon Footprint Awareness Platform is a web-based application designed to educate users about their environmental impact and promote sustainable living practices. The platform provides an interactive carbon footprint calculator that helps users estimate their carbon emissions based on daily activities such as transportation, electricity consumption, and fuel usage.

By increasing awareness of carbon emissions and offering practical recommendations, the platform encourages individuals to make environmentally responsible choices and contribute to a greener future.

## Features

* Interactive Carbon Footprint Calculator
* Carbon Emission Estimation Based on User Inputs
* Sustainability Awareness Content
* Personalized Eco-Friendly Recommendations
* Responsive Design for Mobile and Desktop Devices
* Modern and User-Friendly Interface
* Dark and Light Theme Support
* Environmental Education Resources

## Technologies Used

* React.js
* TypeScript
* Tailwind CSS
* Vite
* Chart.js / Recharts
* Vercel (Deployment)

## Purpose

Climate change is one of the most significant global challenges today. This platform aims to help users understand how everyday activities contribute to carbon emissions and provides actionable insights to reduce their environmental footprint. Through awareness and education, the project supports the goal of building a more sustainable and environmentally conscious society.
